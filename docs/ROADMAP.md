# 🗺️ CORA Roadmap

> Directional and may evolve. Currently **live on Solana devnet**.

---

## ✅ Phase 0 — MVP Live + $CORA Launch · **Live**
*Goal: prove the core loop on-chain and launch the token.*

- Real-time 1v1 logic battles — cards, HP, 10s answer window, damage/heal
- Instant matchmaking + free bot practice (no wallet)
- Blink / private challenges via shareable links
- On-chain escrow + settlement on Solana — winner takes 97.5%
- SOL / BONK / USDC wagers · MagicBlock ER battle state
- 🪙 **$CORA token launch** on Solana
- Live: [coragames.fun](https://coragames.fun) · API + Supabase persistence

## 🔨 Phase 1 — Harden & Content · *Q3 2026*
*Goal: make it sticky and trustworthy before real money.*

- Question bank v2 — new categories (sequence, math, logic, lateral, memory) + difficulty scaling
- Leaderboards, ELO & real wallet history
- Anti-cheat hardening + server-authoritative timing
- $CORA utility v1 — reduced rake when wagering in $CORA
- Reliability: reconnect/resume, settlement retry + observability

## 🚀 Phase 2 — Mainnet Launch · *Q4 2026*
*Goal: real stakes, safely.*

- Security audit of escrow + battle programs (gate to mainnet)
- Mainnet deploy + funded treasury & fee flow
- Squads multisig on upgrade authority
- Mainnet tokens + price-aware wager limits
- $CORA as a wager arena · public launch & onboarding funnel

## 🏆 Phase 3 — Competitive & Social · *Q1 2027*
*Goal: turn matches into a scene.*

- Ranked seasons + ELO tiers, rewards paid in $CORA
- Tournaments & brackets (Blink-powered entry)
- Spectator mode + shareable match recaps
- Friends, rematch, clans / teams
- Backend scale-out — Redis-backed matchmaking to run more than one instance

## 🌐 Phase 4 — Ecosystem & $CORA Utility · *2027+*
*Goal: platform + deepen token value.*

- Mobile / PWA duels
- Community question packs (creator-submitted, $CORA revenue-share)
- $CORA staking + season pass + governance over arenas / rake
- Partner arenas / branded tokens, sponsorships
- Public SDK / Blinks to embed challenges anywhere

---

**Cross-cutting (every phase):** anti-cheat, on-chain fairness & proofs, low latency, and "skill > luck" positioning.
